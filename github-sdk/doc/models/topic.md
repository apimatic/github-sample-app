
# Topic

A topic aggregates entities that are related to a subject.

## Structure

`Topic`

## Fields

| Name | Type | Tags | Description | Getter | Setter |
|  --- | --- | --- | --- | --- | --- |
| `Names` | `List<String>` | Required | - | List<String> getNames() | setNames(List<String> names) |

## Example (as JSON)

```json
{
  "names": [
    "names0",
    "names1",
    "names2"
  ]
}
```
